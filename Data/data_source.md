# Data Source

## Dataset
- **Name:** English Premier League Match Data
- **File:** EPL_2526.csv
- **Season:** 2025/26

## Source
- **Website:** [football-data.co.uk](https://www.football-data.co.uk/englandm.php)
- **Access:** Free, no login required

## How to Download
1. Go to https://www.football-data.co.uk/englandm.php
2. Download the CSV for season 2025/26 (E0.csv)
3. Rename it to `EPL_2526.csv`
4. Place it in this `/data` folder

## Columns Used
| Column | Description |
|--------|-------------|
| HomeTeam | Home team name |
| AwayTeam | Away team name |
| FTHG | Full time home goals |
| FTAG | Full time away goals |
| HS | Home shots |
| AS | Away shots |
| HST | Home shots on target |
| AST | Away shots on target |
